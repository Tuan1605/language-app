## 1. Mistake Auto-review (Ưu tiên cao - Effort thấp)

### Hiện trạng
- `db.mistakes` table đã có, `Mistake` type đã có
- **Đã có onSaveMistake**: QuizView, WritingView, RealExamView
- **CHƯA có onSaveMistake**: ListeningView, DictationView, VocabQuizView, GrammarQuizView, SpeakingView

### Cần làm
Thêm `onSaveMistake` callback vào 5 components còn thiếu:

| File | Logic thêm mistake |
|------|-------------------|
| `src/components/ListeningView.tsx` | Khi用户 chọn sai đáp án listening |
| `src/components/DictationView.tsx` | Khi user nhập sai text (score < 80%) |
| `src/components/VocabQuizView.tsx` | Khi user chọn sai vocab |
| `src/components/GrammarQuizView.tsx` | Khi user chọn sai grammar |
| `src/components/SpeakingView.tsx` | Khi pronunciation score < 80% |

Pattern mẫu (từ WritingView.tsx:61-68):
```tsx
if (score < 80 && onSaveMistake) {
  onSaveMistake({
    id: `mistake-${Date.now()}`,
    type: 'dictation', // hoặc 'question', 'speaking', etc.
    data: lesson,
    wrongAnswer: userAnswer,
    timestamp: new Date().toISOString(),
  });
}
```

Cập nhật `SessionPage.tsx` để truyền `onSaveMistake` cho tất cả views.

## 3. Offline/PWA (Ưu tiên trung bình - Effort trung bình)

### Hiện trạng
- `public/sw.js` có: network-first cho navigations, cache-first cho static assets
- `vite-plugin-pwa` đã config với workbox
- PDF/audio load từ GitHub Releases (external) → KHÔNG được cache

### Cần làm
1. **Cache external assets** - thêm stale-while-revalidate strategy cho GitHub Releases URLs (audio, PDF) trong SW
2. **Manifest improvements** - thêm `prefer_related_applications: false`, splash screen icons
3. **Offline fallback** - thêm trang offline HTML khi mất mạng
4. **Update SW** - bump CACHE_VERSION khi deploy

File sửa: `public/sw.js`
