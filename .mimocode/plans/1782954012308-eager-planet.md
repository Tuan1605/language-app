# Flashcard Phiên âm & Cách đọc

## Mục tiêu
Thêm phần phiên âm (IPA) và cách đọc cho flashcard, hiển thị đầy đủ trên cả 2 mặt của flashcard.

## Hiện trạng
- **Flashcard type** (`src/types/index.ts:3-22`): chỉ có `word`, `definition`, `example`, `imageUrl` — không có phonetic/pronunciation
- **JSON data**: TOEIC 3,000 cards, N2 2,163 cards — format `{id, word, definition, example, topic, imageUrl}`
- **FlashcardView** (`src/components/FlashcardView.tsx`): flip card — front hiển thị word, back hiển thị definition + example
- **contentLoader.ts** (`src/data/contentLoader.ts`): `RawFlashcard` interface chuyển thành `Flashcard`
- **N2 cards**: đã có phonetic nhúng trong word, ví dụ `"研究 (けんきゅう)"`

## Kế hoạch

### Bước 1: Cập nhật Flashcard type
**File**: `src/types/index.ts`
- Thêm 2 field optional vào Flashcard interface:
  ```ts
  phonetic?: string;      // IPA: /nəˈɡoʊʃieɪt/
  pronunciation?: string;  // Cách đọc đơn giản: nuh-GOH-shee-eyt
  ```

### Bước 2: Cập nhật RawFlashcard interface & contentLoader
**File**: `src/data/contentLoader.ts`
- Thêm `phonetic` và `pronunciation` vào `RawFlashcard` interface
- Cập nhật `rawToFlashcard()` để truyền 2 field mới vào Flashcard

### Bước 3: Tạo script generate phonetic data
**File**: `scripts/generate-phonetic.mjs` (mới)
- Đọc flashcards.json hiện tại
- Gọi dictionaryapi.dev (free, không cần API key) cho mỗi từ tiếng Anh
- Extract IPA và tạo pronunciation đơn giản (bỏ dấu /, thay vowel bằng cách đọc)
- Ghi lại flashcards.json với dữ liệu mới
- **Xử lý lỗi gracefully**: nếu API fail cho 1 từ, giữ nguyên và log warning
- Chạy riêng cho TOEIC và N2

### Bước 4: Cập nhật FlashcardView
**File**: `src/components/FlashcardView.tsx`
- **Front side**: Hiển thị word lớn, bên dưới hiển thị `phonetic` (nếu có) và `pronunciation` (nếu có)
- **Back side**: Giữ nguyên definition + example
- Thêm icon loa cho pronunciation để nghe cách đọc

### Bước 5: Cập nhật AddFlashcard
**File**: `src/components/AddFlashcard.tsx`
- Thêm 2 input field cho phonetic và pronunciation (optional)
- Cập nhật `onAdd` interface để truyền 2 field mới

### Bước 6: Cập nhật CollectionView edit modal
**File**: `src/components/CollectionView.tsx`
- Thêm 2 input field vào edit modal cho phonetic và pronunciation
- Cập nhật `editForm` state và `handleSaveEdit`

### Bước 7: Cập nhật AddWordsPage
**File**: `src/pages/AddWordsPage.tsx`
- Cập nhật `CardInput` type để include phonetic, pronunciation
- Truyền 2 field mới khi tạo Flashcard mới

### Bước 8: Reset database & re-seed
- Sau khi cập nhật JSON, cần reset IndexedDB để load dữ liệu mới
- User sẽ cần reset database trong app (nút Reset trong settings)
