# Kế hoạch Cải thiện UI Tất cả Game trong LingoMaster

## Tổng quan

Cải thiện giao diện 10 mini-game + GamesPage hub để đạt mức polished, consistente và visually stunning, giữ nguyên aesthetic neumorphic + Duolingo hiện tại.

---

## Phase 1: Tạo Shared Game Components (nền tảng cho tất cả game)

### 1.1 Tạo `GameShell.tsx` — Wrapper chung cho mọi game
**File mới:** `src/components/games/GameShell.tsx`

Component bọc ngoài mỗi game, cung cấp:
- **Game header bar**: nút back, tên game, badge điểm số/combo
- **Progress bar** trên cùng (hiển thị tiến độ câu/trò chơi)
- **Consistent padding/scroll behavior**
- Props: `title`, `icon`, `score`, `progress`, `difficulty`, `onBack`, `children`

```tsx
<GameShell title="Memory Match" icon={<Brain />} score={matches} progress={matches/requiredPairs} difficulty={difficulty} onBack={onComplete}>
  {/* game content */}
</GameShell>
```

### 1.2 Tạo `GameHUD.tsx` — HUD component chung
**File mới:** `src/components/games/GameHUD.tsx`

Hiển thị thống kê game dạng thanh ngang:
- **Score** (số lớn, màu blue)
- **Combo/ streak** (Zap icon, vàng khi >=3)
- **Timer** (đồng hồ đếm ngược, chuyển đỏ + pulse khi <=10s)
- **Lives** (dãy Heart icons, fill đỏ khi sống, xám khi mất)
- **High score badge** (nhỏ, gold)

Props linh hoạt: `score?`, `combo?`, `timer?`, `lives?`, `maxLives?`, `highScore?`, `timerCritical?`

### 1.3 Tạo `GameOverScreen.tsx` — Màn hình kết thúc game chung
**File mới:** `src/components/games/GameOverScreen.tsx`

Màn hình kết thúc với celebration animation:
- Trophy icon lớn với hiệu ứng scale-in + glow
- Điểm số với counter animation (từ 0 lên số thực)
- EXP earned badge (green pill)
- High score badge (gold pill, chỉ hiện khi có)
- Confetti dots animation (dùng CSS `confetti-dot` có sẵn)
- 2 nút: "Play Again" (btn-duo btn-blue) và "Menu" (btn-duo btn-outline)
- Props: `score`, `expEarned`, `highScore?`, `isWin`, `onRestart`, `onMenu`

### 1.4 Tạo `GameLoading.tsx` — Loading state chung
**File mới:** `src/components/games/GameLoading.tsx`

Skeleton loading consistent cho mọi game:
- 3 skeleton shimmer boxes (dùng class `.skeleton` có sẵn)
- Text "Loading..." với animate-pulse

## Phase 4: CSS Additions

**File sửa:** `src/index.css`

### 4.1 Thêm utility classes mới:
```css
/* Game-specific animations */
@keyframes match-pop { /* scale bounce khi match */ }
@keyframes score-float { /* "+10" bay lên rồi biến mất */ }
@keyframes card-shine { /* shimmer effect trên card */ }
@keyframes shake { /* rung khi sai */ }
@keyframes timer-urgent { /* pulse nhanh khi timer < 10s */ }

.animate-match-pop { animation: match-pop 0.4s ease-out; }
.animate-score-float { animation: score-float 1s ease-out forwards; }
.animate-shake { animation: shake 0.4s ease-out; }
.animate-timer-urgent { animation: timer-urgent 0.5s ease-in-out infinite; }

/* Game card gradient hover */
.game-card-gradient {
  background: linear-gradient(135deg, transparent 0%, var(--blue) / 5 100%);
}

/* Timer bar gradient */
.timer-bar {
  background: linear-gradient(90deg, var(--green) 0%, var(--gold) 50%, var(--red) 100%);
  transition: width 1s linear;
}

/* Neumorphic key press */
.key-pressed {
  box-shadow: var(--shadow-inset-light);
  transform: translateY(2px);
}
```

1. **Dev server**: `npm run dev` → mở `/games` → test từng game
2. **Kiểm tra dark mode**: toggle theme → đảm bảo mọi game vẫn đẹp
3. **Kiểm tra Japanese track**: switch track → đảm bảo color variant hoạt động
4. **Responsive**: test trên mobile viewport (375px) và desktop (1280px)
5. **Test gameplay**: chơi qua từng game → đảm bảo logic game không bị ảnh hưởng
6. **Performance**: kiểm tra không có regression trong animations
